>

## 一、分析师讲解语姿

AI 装作"分析师在带你讲解"，在段落过渡处生成"步骤总结→步骤预告"的桥接句。这是 LLM 注意力机制在段落过渡处的典型行为——它倾向于在转折处复述上一步、预告下一步。

### 步骤承接句式

- `把 X 做完/说完/看完/拆完/理完/弄清/厘清/讲完/聊完/分析完/搞清楚，再/便/就/然后 Y`
- `先看 A` / `接着看 B` / `看完了 A 再说 B`
- `把 A 理清之后` / `厘清了背景`
- `上面说的是 X，下面说 Y`
- `前面分析了 A，现在转向 B`

❌ `把背景厘清之后，我们来看具体案例。`
✅ `背景就是这样。下面是具体案例。`（或直接讲案例，不要预告）

❌ `上面说的是供给侧，下面说需求侧。`
✅ 删掉这句，直接开始讲需求侧。

### 改法

删掉步骤承接的桥接句，直接讲下一步的内容。段落之间不需要"承上启下"的导游词——内容本身的逻辑会连接。

## 二、戏剧化揭露修辞

AI 用"撕下面具""扯下遮羞布"的戏剧化修辞包装普通分析，假装在揭露真相。

### 遮蔽物名词

遮羞布 / 最后一块遮羞布 / 面具 / 画皮 / 伪装 / 外衣 / 皇帝的新衣 / 幌子 / 烟幕弹

### 揭露动作 + 遮蔽物

- `扯下/撕下/揭下/摘下/掀开 + 遮羞布/面具`
- `剥开/戳穿/戳破/揭穿/撕开/揭开 + 表象/真面目/本质/真相/外壳/包装`
- `把 XXX 的遮羞布扯下来`
- `剥开 XXX 的外衣`
- `撕下 XXX 的面具`
- `戳穿了 XXX 的谎言`
- `不过是一块遮羞布`

### 数据展示动作（低戏剧度副本）

- `数字摆/数据摆/事实摆/账目摆 + 出来/在那里/到桌面上`
- `摆得清楚/摆得更清楚/摆得一清二楚`
- `再摆一组数据`

### "下去"动作拖尾壳

- `切下去/砍下去/压下去/打下去/推下去/落下去/拍下去`（配政策/法案/预算/处罚抽象主语）

❌ `这不过是一块遮羞布，撕下来里面是利益的算计。`
✅ 说清楚利益算计是什么，不要用"遮羞布"包装。

### 改法

删掉戏剧化修辞，直接讲分析内容。真相不需要"撕面纱"的仪式感——直接说是什么就是什么。

## 三、极值判断句式

AI 用"最 X""真正可怕的是"做极值判断，制造冲击力。

### 泛化"最 X"

- `最……的地方在于/是`
- `最先/最早/最容易/最常/最该 + 往往/反而/还是`
- `最先能用上的，往往还是……`

### 真正 X 的是

- `真正可怕的是` / `真正残酷的是` / `真正讽刺的是` / `真正危险的是` / `真正荒谬的是`
- `残酷之处在于` / `讽刺之处在于` / `荒谬之处在于` / `吊诡之处在于` / `可怕之处在于`
- `更残酷的是` / `更可怕的是` / `更讽刺的是`

### 解释预告壳 + 工整递进

- `原因/答案/问题/逻辑/现实 + 很/非常/相当 + 简单/朴素/清楚/直接/残酷/现实`
  - `原因很朴素` / `答案很简单` / `现实很残酷`
- `原因/答案/事实/现实 + 摆在那里/摆在眼前`
- `A 尚且……，B 更/更是……`
  - `职场天花板尚且压在头上，街头的代价更直接`

❌ `真正可怕的不是技术本身，而是人对技术的依赖。`
✅ `人对技术的依赖比技术本身更值得担心。`（或直接讲依赖的问题，不要用"真正可怕的是"开场）

### 改法

删掉极值判断的包装词，直接说判断内容。"真正可怕的是 X" → "X"。"最 X 的地方在于" → 直接说 X。

## 四、伪学术与政论媒体腔

AI 装作在做学术分析或政论评论，堆砌术语和机构名做权威铺垫。

### 伪学术场域词

- `舆论场` / `话语场` / `权力场` / `博弈场` / `公共场域` / `意义场域`
- `宏大叙事` / `国家叙事` / `官方叙事` / `主流叙事` / `元叙事`
- `摊开来看` / `摊开来说` / `把 XXX 摊开来`
- `初登台` / `正式登台` / `粉墨登场`
- `面子上是 X，里子是 Y` / `给足面子、掏空里子`
- `底层逻辑` / `顶层设计` / `认知升级` / `降维打击` / `赛道` / `红利` / `存量` / `增量` / `博弈` / `范式` / `路径依赖` / `权柄` / `话语权`

### AI 高频隐喻词（比喻义）

噪音 / 背景噪音 / 信号 / 底色 / 光谱 / 滤镜 / 解药 / 土壤 / 拼图 / 镜像 / 路径 / 尺子 / 标尺 / 横截面 / 切面 / 切片 / 透镜 / 棱镜 / 窗口 / 缩影 / 账本 / 成绩单 / 清单

> 尤其禁止把复杂问题写成可结算的账本、可打分的成绩单、可勾选的清单。

### 政论 / 媒体研究术语

- `口径` / `统一口径` / `对齐口径` / `通稿口径` / `官方口径`
- `定调` / `给 X 定了调` / `调子定下来` / `调门` / `起调`
- `话术` / `这一套话术` / `一整套话术`
- `通稿` / `通稿写法` / `像通稿` / `念通稿` / `喉舌`

### 机构串联做权威铺垫

- `X 研究部和 Y 咨询部`
- `X 大学和 Y 智库`
- `X 银行和 Y 基金会`

❌ `在当前的舆论场中，这套话术早已扎根，成为主流叙事的底层逻辑。`
✅ 说清楚是谁在说什么，不要用"舆论场""话术""底层逻辑"打包。

### 改法

删掉伪学术和政论术语，换成具体的人、事、话。如果真在写学术论文，这些词有准确含义时可以留——但公号文/评论文里堆这些就是 AI 味。

## 五、标签讽刺与群体对立化指代

AI 用标签把一群人打包，制造"我们对他们"的对立感。

### 万能标签

万能贴纸 / 万能标签 / 万能挡箭牌 / 万能护身符 / 万能解药 / 万能钥匙 / 万能借口 / 万能筐 / 万能背锅侠 / 万能帽子

### 群体标签 + 起劲/响/高/凶

- `骂得最起劲的那批 X`
- `喊得最响的那群 X`
- `跳得最高的那波 X`
- `闹得最凶的一拨 X`
- `表态最积极的那批 X`
- `转发最快的那群 X`

### 群体指代

- `那批海外华人` / `那群键盘侠` / `那拨自媒体` / `那帮键政人` / `那群小镇做题家`

### 行为反差套式

- `平日骂 X 骂得最起劲，结果自己 Y`
- `平时高喊 X，事到临头自己 Z`

❌ `骂得最起劲的那批人，事到临头跑得最快。`
✅ 说具体是谁、具体做了什么。

### 改法

拆掉标签，说具体的人和事。"那批 X" → 具体是谁。"骂得最起劲" → 具体骂了什么。

## 六、接力比喻

AI 用"接力棒""火炬"比喻代际/历史责任，制造庄严感。

- `接力棒` / `这一棒` / `那一棒` / `火炬` / `接力` / `接力跑` / `末班车`
- `历史的接力` / `时代的接力`
- `接过 X 的棒` / `把棒交给 X` / `把火炬传给 X`
- `这一棒传到 X 手里` / `这一棒掉在 X 这里`
- `X 不必接这一棒` / `X 不必背这一棒`（豁达祈使）

❌ `历史的接力棒传到了这一代人手里，他们必须跑好这一棒。`
✅ 说清楚这一代人面对的具体问题是什么。

### 改法

删掉接力比喻，直接说责任/任务的内容。

## 七、分析包装三件套

AI 用"一整套 X"把零散观察包装成系统分析，配"扎根/落地/生根"假装有机制。

### 一整套

`一整套话术` / `一整套机制` / `一整套说辞` / `一整套逻辑` / `一整套套路` / `一整套叙事` / `一整套规则` / `一整套说法`

### 扎根/落地

- `话术在 X 里扎根`
- `套路在 Y 中落地`
- `这套说辞在 Z 里生根`
- `观念在 W 里蔓延`
- `论调站住脚`
- `思路扎下根来`

### 之所以

- `X 之所以在 Y 中 Z`
- `X 之所以能在 Y 里流行`
- `X 之所以站住脚`

❌ `这一整套话术在舆论场里扎根，之所以能站住脚，是因为它切中了某种焦虑。`
✅ 说清楚具体是什么话术、谁在说、为什么有人信。

### 改法

拆掉"一整套"包装，说具体是哪几句话。"扎根/落地"删掉，换成具体在哪些场景出现。

## 八、各种"壳"

AI 的高频句式骨架——换词逃不掉的"壳"。

### 8.1 重复问题判词与"并不 X"降调壳

- `问题不在于……。问题在于……`（连续两句以"问题"开头）
- `X 多了去了`
- `并不复杂/并不新鲜/并不意外/并不奇怪/并不难懂/并不高明/并不深奥`

❌ `问题不在于技术。问题在于人。`
✅ `问题出在人身上，和技术无关。`

### 8.2 "省"字解释捷径

- `省事/省力/省心/省麻烦/省去了/省掉了/省下了`
- `省得再/省得去`
- `替代思考`

❌ `这么做省事，不用动脑子。`
✅ 说清楚具体省了什么、为什么这样更简单。

### 8.3 "却"字工整反差壳

- `字面上/表面上/名义上/看似 + X，却 + Y`
- `谐音/背后/里面/暗处 + 藏着/带着 + Y`

❌ `表面上是为客户好，却藏着算计。`
✅ `说是为客户好，实际是在算计。`

### 8.4 "很容易"解释捷径

- `很容易 + 把/被/让/想到/变成/说成/理解成/归为/滑向`
  - `人们很容易把 X 理解成 Y`
  - `这种说法很容易滑向……`

❌ `人们很容易把它理解成阴谋。`
✅ 说清楚为什么有人会这样理解，不要用"很容易"打包。

### 8.5 "想象中"稻草人壳

- `某些人/有些人/很多人/不少人 + 想象/以为`
- `想象中那么/想象里那样`
- `本来就没有……想象中那么……`

❌ `没有你想象中那么复杂。`
✅ 说清楚实际有多复杂。

### 8.6 温和让步壳与"X 的地方"包装壳

- `X 当然/自然 + 有 + 它/他/她/其 + 的 + 道理/逻辑/来由/来处/原因/苦衷/考量/合理性`
  - `这种安稳当然有它的人情来处`
- `X 的麻烦/问题/争议 + 原不该/原本不该/本不该 + 被 + 拖进/卷入/牵扯/裹挟`
  - `马兜铃酸的麻烦，原不该被拖进文化身份的争执里`
- `X 还有一个/另一个 + 更/最/真正 + 麻烦/棘手/讨厌/复杂/严重/微妙/吊诡 + 的地方`
- `X 麻烦/棘手/难办的地方在于/是`

### 8.7 指示性定位收束壳

- `主语 + 正/就/恰好/偏偏/眼下/此刻 + 卡/落/夹/陷/停/堵/挤/吊/悬/绕/绊 + 在 + 这条/这道/那个 + 裂缝/缝隙/夹缝/边界/线/点/口 + 里/上/中`
  - `中药注射剂正卡在这条裂缝里`
- `X 的 + 麻烦/难处/要害/症结 + ，+ 正/就/恰恰/偏偏 + 在这里`
  - `中药注射剂麻烦的地方，正在这里`

### 8.8 无生命对象拟人化否定壳

AI 让无生命对象"不认/不在乎/不分"抽象概念，制造冷酷事实感。

- `不/不会/不肯/不愿/听不懂/看不见/记不住/不在乎/不认/不挑/不分/不等/不讲/不管/不理 + 抽象对象`
- 主语：`血管/身体/细胞/器官/伤口/心脏`、`分子/化学/物理/药理/病毒/细菌/基因/DNA`、`数据/数字/账本/曲线/报表`、`历史/时间/市场/算法/制度/规则/法律/河流/山/路`
- 宾语：`这些词/那些话/这套说法/你那一套/立场/情绪/标签/口号/道理/面子/身份`
- 完整模板：`血管不认这些词` / `市场不在乎你那一套` / `数据不挑立场` / `分子不分中西医` / `历史不会等任何人` / `算法不讲情面` / `病毒不看护照`

> 变体逃逸警告：LLM 会把"血管"换成"细胞/分子/数据/历史"做主语替换逃逸。抓句式骨架（无生命主语 + 否定动词 + 抽象概念宾语），不要只抓具体词。

❌ `市场不在乎你那一套说辞。`
✅ 说清楚市场实际按什么逻辑运行。

### 改法

各种"壳"的共同改法：拆掉句式骨架，换普通句式说同样的话。"表面 X 却 Y" → "说是 X，实际 Y"。"X 当然有它的道理" → 删掉或直接说道理是什么。"市场不在乎" → 说市场实际怎么运行。

## 九、升华训诫结尾与金句叠句

中文论说文最典型的 AI 收尾腔——也是人类读者最容易"感觉"到 AI 味的地方。

### 呼告式升华

- `愿我们都能做出正确的选择`
- `愿我们……`
- `让我们都能……`
- `让我们一起……`

### 神圣化

- `这项神圣的……`
- `不可磨灭的印记`
- `历史将会记住……`

### 排比金句

- `每一……都是……，每一……都是……`
  - `每一张选票都是责任，每一次参与都是担当`
- `我们是选择了 X 还是 Y`
  - `我们是选择了清醒还是糊涂，是勇气还是妥协`

### 制造主题金句并反复引用

AI 造一句口号（`经济为政治之基石，军事乃政治之延伸`），然后在文章里反复引用它——这是最明显的 AI 讲义腔。

### "为……作出更大贡献"

- `为本国发展与世界和平作出更大贡献`
- `并在其中找到自己的位置`

❌ `愿我们都能在时代的洪流中，找到属于自己的位置，为这个世界贡献自己的一份力量。`
✅ 收在具体事实或克制判断。文章讲的是什么问题，结尾就停在那个问题上。

### 改法

不要为了制造力量临时造一句口号再反复引用。收尾可以停在原文已有的具体事实或克制判断上，也可以直接结束；作者原本有意设计的主题句、品牌语或叙事回声则按功能保留。

## 十、抽象具象路线反覆回指

AI 把抽象过程具象成"一条线/路线/轴线/脉络/桥"，然后反复回指"这条线/这条路线/这条路"，配生长/移动动词。

### 三信号必须叠加才算命中

1. **具象化**：抽象过程被写成 `一条线/路线/轴线/脉络/桥/路`
2. **反覆回指**：`这条线/这条路线/这条路` 在后文反复出现
3. **生长/移动动词**：`走出来/长出来/铺出来/重新长出来/延伸/串起来/拓宽`

三个信号同时出现才算 AI 味。单独一个不算——`这条路走不通`、`升迁阶梯只有一条路` 是地道中文惯用语，不要动。

❌ `这条路是从改革开放开始走出来的，一路延伸到现在，串起了几代人的记忆。后面这条路还在拓宽，未来还会继续长下去。`
✅ 说清楚具体的进程，不要用"一条线"打包然后用生长动词反复回指。

### 改法

拆掉具象化路线，换回抽象过程的具体描述。删掉反覆回指的"这条线/这条路"，每处换成具体指代的内容。

## 变体归并：新口癖归入已有族

遇到新的口癖，不要无限往清单加词。归入已有的问题族：

| 族 | 特征 | 例子 |
|---|---|---|
| 调试腔 | 用 SRE/工程师口吻说普通事 | 稳稳兜住、收口、根因、落盘 |
| 庸医问诊腔 | 装作诊断专家 | 抠出来、揪出来、我不猜 |
| 暴力动作腔 | 用激烈动词说普通操作 | 补一刀、更狠、打掉 |
| 主动出击腔 | AI 过度主动邀功 | 我已确认、要不要我、顺手 |
| 总结提示腔 | 念讲义式总结 | 一句话总结、结论先说、省流版 |
| 过度接住腔 | 咨询式安抚 | 稳稳接住你、你只是…… |
| 郑重预告腔 | 身份认证式夸奖 | 我必须很认真地说、你问到了核心 |
| 伪通俗腔 | 装作在说大白话 | 说白了就是、说人话就是、翻译成大白话 |
| 话题框架腔 | 用框架词打包话题 | 关于 X 这件事、X 这个事情的核心 |

新口癖归入对应族，按族的处理方式改。
