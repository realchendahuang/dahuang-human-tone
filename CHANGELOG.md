# Changelog

本项目使用语义化版本号记录 Skill 的重要变化。

## 2.0.0 — 2026-07-10

- 重写 Skill 入口，按改写、诊断和轻改三种模式执行。
- 改成按场景懒加载 reference，不再要求所有文本执行形状量化。
- 删除没有可靠依据的固定句长、段长和词频阈值。
- 把“AI 判定”收紧为可观察的写作诊断，明确禁止来源归因和伪精确评分。
- 将事实保护从“一字不改”修正为“含义与身份不变”，减少对正常句法调整的阻碍。
- 增加长 reference 的目录，并清理法证式“定罪”措辞。
- 对齐 Codex UI 元数据，只保留受支持字段。

- 修正手动安装路径，明确真正的 Skill 包位于 `skills/dahuang-human-tone/`。
- 对齐仓库 README 与 Skill 包 README。
- 优化 Skill 描述和 Codex UI 元数据。
- 补充 MIT License。
