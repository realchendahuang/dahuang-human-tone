# 改写示范

> 同一段文本，去味前后的对比。每段标明删了什么壳、改了什么结构、注了什么人味。
>
> 覆盖五个场景：公号文、技术文、小红书、小说、通用。

---

## 示范一：公号文 / 随笔

### 改前（AI 味）

> 在当今快速发展的数字时代，人工智能正在深刻地改变着我们的生活方式。值得注意的是，这不仅仅是一次技术变革，更是一次思维方式的革命。深入探讨这一问题，我们会发现，真正的挑战不在于技术本身，而在于人对技术的理解。
>
> 首先，我们需要认识到技术的边界。其次，我们要学会与机器协作。最后，保持独立思考的能力至关重要。只有真正理解了这一点，我们才能在 AI 时代找到属于自己的位置。
>
> 综上所述，AI 不是要取代人类，而是要赋能人类。让我们一起拥抱这个充满无限可能的未来。未来可期。你觉得呢？

### 改后（人味）

> AI 在改我们做事的方式，这件事没什么悬念。有意思的是，真正的挑战不在技术——技术就是工具，工具会越来越好——而在于人怎么理解它。
>
> 技术有边界，得知道它在哪停。跟机器协作是一门手艺，得练。独立思考的能力别丢——这话说了一百遍了，但在 AI 面前它突然又重要了起来。
>
> AI 不会取代人，但它会把人分成会用和不会用两类。具体怎么分，现在还说不好。

**删了什么**：
- `在当今快速发展的数字时代`（开场路标）
- `值得注意的是`（路标词）
- `这不仅仅是……更是……`（否定式排比）
- `深入探讨`（伪口语化动作词）
- `首先……其次……最后……`（机械顺序壳）
- `至关重要` / `只有……才`（渲染词 + 关联句式）
- `综上所述`（收尾路标）
- `不是……而是……`（二分对照壳）
- `赋能`（互联网黑话）
- `让我们一起拥抱……未来可期。你觉得呢？`（升华训诫收尾 + 假互动）

**注了什么人味**：加了第一人称判断（"没什么悬念""有意思的是"）、承认不确定（"现在还说不好"）、收在克制判断上而不是升华。

## 示范二：技术文 / 教程

### 改前（AI 味）

> 值得注意的是，在部署此功能之前，确保您的环境已满足以下要求。此外，本功能依赖于 Redis 7.0 及以上版本，以提供更强大的缓存能力。接下来，我们将深入探讨如何配置缓存参数。
>
> - **连接池配置：** 连接池配置通过优化参数得到显著增强。
> - **超时设置：** 超时设置通过合理调整提升了系统的稳定性。
> - **重试机制：** 重试机制通过自动恢复确保了服务的可用性。

### 改后（干净）

> 部署前确认环境要求。本功能依赖 Redis 7.0+。
>
> 连接池、超时、重试三个参数需要调：连接池大小按并发量设，超时建议 3 秒，重试次数别超过 3 次否则会放大故障。

**删了什么**：
- `值得注意的是` / `此外` / `接下来我们将深入探讨`（路标词 + 协作腔）
- `确保您的` / `以提供更强大的`（渲染词 + 系动词回避）
- `**X：**Y` 冒号模板（三处粗体标题列表）
- `通过优化/合理调整/自动恢复` + `显著增强/提升稳定性/确保可用性`（-ing 肤浅分析 + 宣传腔）

**没注人味**：技术文不需要注入灵魂。中性就是对的。只删协作腔和路标词，保留技术信息。

## 示范三：小红书 / 社媒

### 改前（AI 味）

> 姐妹们！今天来给大家拆解一下我是怎么用 AI 工具提升工作效率的！🚀
>
> 不得不说，自从用了这些工具，我的效率直接拉满！具体来说，我主要用了以下几个工具：
>
> - **ChatGPT：** 帮我写文案，深入探讨各种话题
> - **Midjourney：** 赋能我的设计工作，充满创造力
> - **Notion AI：** 梳理我的知识体系，打造个人闭环
>
> 总的来说，AI 工具不仅仅是提高效率的手段，更是改变思维方式的力量。你有没有类似的经历？在评论区聊聊吧！✨

### 改后（地道）

> 分享几个我天天在用的 AI 工具。
>
> ChatGPT 写文案，但得自己改，直接用会被看出来。Midjourney 出图快，但手和文字还是不行。Notion AI 整理笔记凑合，复杂的不指望它。
>
> 工具是好的，但别指望它们替你思考。

**删了什么**：
- `姐妹们！` / `🚀` / `✨`（emoji + 称呼腔——少量用是地道，堆叠是 AI）
- `拆解一下`（伪口语化动作词）
- `不得不说` / `具体来说`（路标词）
- `拉满` / `深入探讨` / `赋能` / `充满创造力` / `梳理` / `打造` / `闭环`（互联网黑话 + 渲染词）
- `**X：**Y` 冒号模板
- `总的来说` / `不仅仅是……更是……`（收尾路标 + 否定式排比）
- `你有没有类似的经历？在评论区聊聊吧！`（假互动）

**注了什么人味**：给了具体的使用体验（"得自己改""手和文字还是不行""复杂的不指望它"）——这是真实使用者才会说的细节。收在克制判断上（"别指望它们替你思考"）。

## 示范四：小说 / 故事

### 改前（AI 味）

> 他不禁微微一笑，嘴角微扬，心中暗道这个人不简单。缓缓说道："你的提议很有意思。"
>
> 然而，他深知，这不仅仅是一次简单的合作，更是命运的开端。在这个充满变数的时代，每一次选择都至关重要。他不禁陷入了沉思。

### 改后（有声音）

> 他笑了一下。不是那种客套的笑，是那种你看到对手出了好牌时的笑。
>
> "你的提议有点意思。"
>
> 他说完没再接话，端起杯子喝了一口。合作的事他得想想——不是想要不要做，是想怎么做才不出事。

**删了什么**：
- `不禁` / `微微` / `缓缓`（小说套话——不承载声音的修饰词）
- `嘴角微扬` / `心中暗道`（AI 小说腔）
- `然而` / `深知`（路标词 + 渲染词）
- `这不仅仅是……更是……`（否定式排比）
- `在这个充满变数的时代`（开场路标）
- `至关重要`（渲染词）
- `不禁陷入了沉思`（套话收尾）

**注了什么人味**：给了具体的笑的质感（"不是客套的笑，是看到对手出好牌时的笑"）、给了具体动作（"端起杯子喝了一口"）、内心独白给了具体内容（"不是想要不要做，是想怎么做才不出事"）而不是"陷入了沉思"。

## 示范五：通用 / 商务

### 改前（AI 味）

> 感谢您抽出宝贵的时间与我们交流。您的反馈对我们至关重要，我们将认真对待每一条建议。值得注意的是，我们已经将您的意见同步给相关团队，并将在后续版本中持续优化。
>
> 如果您有任何其他问题，请随时与我们联系。希望这次交流对您有所帮助。让我们一起期待更好的产品体验！

### 改后（直接）

> 收到反馈，已经转给产品团队了。下个版本会改。
>
> 有问题随时找我。

**删了什么**：
- `感谢您抽出宝贵的时间` / `您的反馈对我们至关重要`（谄媚腔）
- `我们将认真对待每一条建议`（宣传腔）
- `值得注意的是` / `并将在后续版本中持续优化`（路标词 + -ing 肤浅分析）
- `如果您有任何其他问题，请随时与我们联系`（协作痕迹）
- `希望这次交流对您有所帮助`（协作痕迹）
- `让我们一起期待更好的产品体验`（升华收尾）

**没注人味**：商务回复不需要注入灵魂。直接、清楚就够了。删协作腔和谄媚腔，保留信息。

---

## 改写规律总结

从五个示范里能看出几个规律：

1. **公号文/随笔删得最多**——路标、壳、升华全删，还要注人味
2. **技术文删得最少**——只删协作腔和路标词，保留信息，不注人味
3. **社媒要小心**——删太多会改成说明书，保留口语感和具体体验
4. **小说靠细节**——删套话，加具体动作和内心独白的质感
5. **商务回复最简单**——删谄媚和协作腔，直接说事
6. **所有场景的共同点**——删路标词、删假互动、删升华收尾，这三样在哪都要删
